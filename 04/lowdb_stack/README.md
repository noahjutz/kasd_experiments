# Lowdb with the KASD Stack

## Stack

|           |                         |
| --------- | ----------------------- |
| Framework | Hapi + Plugins          |
| Frontend  | Handlebars + Bulma      |
| Backend   | Lowdb                   |
| Style     | ESLint + Prettier       |
| Test      | Mocha TDD + Chai Assert |

## Installation

```sh
npm i @hapi/hapi @hapi/vision handlebars lowdb
npm pkg set type=module # ES Module
```
